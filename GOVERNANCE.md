# Project Governance

## Project Overview
NewLandingPage is a personal professional portfolio project maintained by Diego Maury. Contributions are welcome when they improve clarity, performance, accessibility, or deployment quality, but final decision-making remains centralized to preserve a coherent public-facing narrative and visual standard.

## Maintainer
**Primary Maintainer**: Diego Maury
- **Email**: Not publicly declared in this repository
- **Timezone**: Maintainer timezone is managed privately
- **Availability**: Best response times are reflected through issue and pull request activity

## Contribution Guidelines
### How to Contribute
1. **Issues**: Open issues for defects, copy problems, accessibility gaps, or improvement ideas
2. **Pull Requests**: Small, focused improvements are preferred over broad redesigns
3. **Discussions**: Use issues or repository discussion threads to propose structural or content changes first

### Contribution Process
1. Open an issue or proposal for non-trivial changes
2. Create a focused branch
3. Keep edits aligned with the site positioning, performance constraints, and bilingual content rules
4. Submit a pull request with screenshots or before/after context when UI changes are involved
5. Wait for maintainer review before merge

## Decision Making
As the primary maintainer, Diego Maury makes final decisions regarding:
- Feature additions and removals
- Architecture changes
- Release schedules
- Code quality standards

### Community Input
Community input is useful, especially for portfolio clarity and usability:
- Major changes should be discussed before implementation
- Breaking content or structure changes should explain impact on brand positioning
- Suggestions are evaluated against simplicity, credibility, and maintenance cost

## Communication
- **Primary**: GitHub Issues and Pull Requests
- **General Discussion**: Repository discussions or issue threads tied to a concrete proposal
- **Direct Contact**: Use private maintainer contact channels for sensitive matters

## Project Goals
- Position NewLandingPage as Diego Maury's professional portfolio with clear positioning and measurable outcomes
- Keep the site fast, portable, and simple to deploy on GitHub Pages
- Maintain high readability, responsive behavior, and strong content quality
